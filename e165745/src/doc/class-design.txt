・プレイヤーの初期位置をどう保存したら良いか？（プレイヤーが迷路に入った際の初期位置であって、現在位置ではない。入り口と考えよう）
    プレイヤーの初期位置は、デフォルトの0行目の値を取り出すことで保存する。
    取り出すので、0行目以降は繰り上げになるようにする。(そうすることで、迷路マップが行列の値がそのまま引用できるようになる。)←迷路情報で補足

・迷路サイズをどう保存したら良いか？
　   プレイヤーの初期位置と同様に、取り出すことで保存する。

・迷路情報をどう保存したら良いか？
    プレイヤーの初期位置と、迷路サイズを取り出すことで保存しているので、
0行目    #######
1行目    # s   #
2行目    # #####
3行目    #   #G#
4行目    # ### #
5行目    # ### #
6行目    #     #
7行目    #######
                となるように、出力されるので、デフォルトと違い行の値がずれていない。
                よって、これを保存する。

・map.txtは一例である。同じ書式で記載された他迷路であっても同様に読み込めるようにするにはどうしたら良いか？
    同じ書式であるならFile名を変えるだけでできる。だが、そのたびにスクリプトを変更するのは面倒なので、
    ファイル名を入力するタイプに変更する。

・アクセサ（getter/setter）は必要だろうか？
    アクセサは隠蔽化に使うので、迷路情報を隠蔽する必要があるなら必要。
