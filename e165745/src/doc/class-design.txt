・プレイヤーの初期位置をどう保存したら良いか？（プレイヤーが迷路に入った際の初期位置であって、現在位置ではない。入り口と考えよう）
    プレイヤーの初期位置は、デフォルトの0行目の値を取り出すことで保存する。
    取り出すので、0行目以降は繰り上げになるようにする。(そうすることで、迷路マップが行列の値がそのまま引用できるようになる。)←迷路情報で補足

・迷路サイズをどう保存したら良いか？
　   プレイヤーの初期位置と同様に、取り出すことで保存する。

・迷路情報をどう保存したら良いか？
    プレイヤーの初期位置と、迷路サイズを取り出すことで保存しているので、
0行目    #######
1行目    # s   #
2行目    # #####
3行目    #   #G#
4行目    # ### #
5行目    # ### #
6行目    #     #
7行目    #######
                となるように、出力されるので、デフォルトと違い行の値がずれていない。
                よって、これを保存する。

・プレイヤーの現在位置をどう保存したら良いか？
    初期状態のプレイヤー座標を変数に保存し,プレイヤーが移動した座標分変数に加減することで保存できる。

・歩数をどう保存したら良いか？
    プレイヤーの移動をfor文やif文で回し,繰り返すたびに変数に+1ずつしていくことで歩数が保存できる。


・迷路内を移動するにはどのような機能が必要だろうか？
    壁(#)に進むことができないようにする。
    行き止まり(壁)ならば,分岐点まで戻る。

アクセサ（getter/setter）は必要だろうか？
   データの隠蔽が必要なので必要である。

他にプレイヤークラスが果たすべき機能はなんだろうか？
    一度行った道が意味止まりだった場合,元の場所に戻った後,その道に行かないようにする機能。
